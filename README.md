# Digitizer Regressor

A web app for digitizing graphs and images, calibrating X and Y axes, capturing data points, and calculating linear regressions.

## Features

- Upload images of graphs
- Calibrate axes with reference points
- Multiple regression lines per image
- Click to capture data points
- Linear regression with R² coefficient
- Export data to CSV

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Bun runtime

## Setup

```bash
bun install
bun run dev
```

## Usage

1. Upload an image
2. Click to set X1, X2, Y1, Y2 calibration points
3. Add lines and capture points
4. Calculate regressions
5. Export CSV
